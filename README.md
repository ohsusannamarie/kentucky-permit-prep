# Dakota’s Kentucky Permit Prep

A mobile-first, ADHD-friendly study app built to help Kentucky permit applicants prepare for the written knowledge test.

**Live App:** https://ohsusannamarie.github.io/kentucky-permit-prep/

## What It Includes

### Practice Questions

* 162 Kentucky-specific practice questions
* Detailed explanations for every answer
* Study by topic or mixed review
* Frequency indicators showing how often questions appear on the exam
* Review missed questions after each practice session

### Mock Permit Exam

* 40-question exam format
* 30-minute timer
* 80% passing score (32/40)
* Detailed answer review and explanations

### Flashcards

* 50 flashcards across 8 Kentucky permit-test categories
* Filter by category
* Mastery tracking
* Mobile-friendly card interface

### Visual Learning Tools

* Right-of-Way Trainer with 20 interactive scenarios
* Lane Markings Guide with visual examples and quiz
* Road Sign Reference with diagrams and explanations

### Study Support

* Beginner Mode for learners starting from zero
* Study Plans (1, 3, 7, and 14 days)
* Progress Tracking and Readiness Score
* Test Day preparation checklist
* Kentucky permit process guide
* Learning videos organized by topic

## Designed for Real Learners

This app was built around a simple idea:

Most permit-prep resources are either overloaded with ads, generic across multiple states, or assume learners already know the basics.

Kentucky Permit Prep focuses on:

* Clear explanations
* Visual learning
* Short study sessions
* Mobile-first design
* ADHD-friendly organization
* Building understanding instead of memorization

## Kentucky Permit Test Facts

* 40 questions on the written knowledge test
* 32 correct answers required to pass (80%)
* Vision requirement: 20/40
* Adult BAC limit: 0.08%
* Under-21 BAC limit: 0.02%
* Signal before turning: 100 feet
* Minimum following distance: 4 seconds
* High beam dimming: 500 ft oncoming, 300 ft following

Always verify current testing requirements through the Kentucky Transportation Cabinet and Kentucky State Police.
